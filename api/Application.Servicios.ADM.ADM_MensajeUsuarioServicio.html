Class ADM_MensajeUsuarioServicio

El servicio ADM_MensajeUsuarioServicio es el encargado de gestionar los mensajes que se envían a los usuarios. Envía un mensaje, lo marca como leído o lo elimina. Obtiene una lista de mensajes de un usuario específico.
