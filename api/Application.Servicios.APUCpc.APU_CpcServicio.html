ActualizarCpcEquipoAsync(Int32, String, String, String, String)

Actualiza o inserta la información de un equipo específico en la base de datos, basándose en su identificador, dentro de CPC Proyecto.

Declaration
public async Task<string> ActualizarCpcEquipoAsync(int EquipoId, string Codigo, string Nombre, string Tipo, string Porc)
Parameters
Type Name Description
System.Int32 EquipoId

Identificador del equipo que se desea actualizar o insertar
System.String Codigo

Código CPC del equipo. Si se proporciona un valor, este se actualizará en la base de datos
System.String Nombre

Nombre CPC del equipo. Si se proporciona un valor, este se actualizará en la base de datos
System.String Tipo

Tipo CPC del equipo. Si se proporciona un valor, este se actualizará en la base de datos
System.String Porc

Porcentaje CPC del equipo. Si se proporciona un valor, este se actualizará en la base de datos
Returns
Type Description
System.Threading.Tasks.Task<System.String>

Un mensaje de confirmación que indica que el equipo con el identificador proporcionado a sido actualizado correctamente.
Exceptions
Type Condition
System.InvalidOperationException

Lanza una excepción si:

  • Lanza una excepción si el identificador `EquipoId` proporcionado es cero
  • El equipo con el identificador proporcionado no se encuentra
  • El valor proporcionado para el porcentaje no es un número decimal valido.
